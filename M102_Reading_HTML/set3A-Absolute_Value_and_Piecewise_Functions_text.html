MAT-102: 3A-Absolute Value and Piecewise Functions

This sections starts with a review of absolute value and then extends the concept to a larger class of functions called "piecewise" functions. Objectives:
  • Solve absolute value equations.
  • Identify and understand translations of absolute value functions.
  • Graph and interpret absolute value and piecewise functions.
  • Determine the domain and range of piecewise functions.

$\S$ 3A.1 Absolute Value Functions

There are a few ways to describe what is meant by the absolute value $|x|$ of a real number $x$. You may have been taught that $|x|$ is the distance from the real number $x$ to $0$ on the number line. So, for example, $|5| = 5$ and $|-5| = 5$, since each is $5$ units from $0$ on the number line.

Note that the distance from zero is a positive number and the absolute value function is often thought of as meaning “whatever comes in, a positive number comes out.” This notion works if you are only considering taking the absolute value of a number: $| 5 |= 5$ and $| -5 |= 5$. But it's a little too simplistic if we consider the absolute value as a function. For functions, we need to keep the "distance" idea firmly in mind.

Consider the function that gives the distance between a number and $3$. Here we are shifting the measurement of distance from zero to $3$, a shift of $3$ units to the right. The new "translated" function is $g(x) = |x-3|$. We see that $g(-5) = |-5-3| = |-8| = 8$ and $g(5) = |5-3|=|-2|=2$. $\checkmark$

So, $f(x) = |x|$ is the function that gives the distance between $x$ to $0$. The domain of this function consists of all real numbers, but the range is $[0,\infty)$ since the distance is nonnegative. When $x \geq 0$, $f(x) = x$ and the graph of this function is a line with slope $1$ and vertical intercept $0$. When we graph values for negative $x$'s the function returns positive values. Mathematically, one way to change the sign on a number is to multiply it by $-1$, so in essence $f(x) = -x$ if $ x \lt 0$. This part of the graph, the section to the left of zero, is a line with slope $-1$ and intercept $0$.

Now that we have the basic shape of the absolute value function, graphing translations of the function should be fairly straight forward.

Example 3A-1: Translations of the absolute value function

The function $g(t) = |x-2| + 3$ is just the absolute value shifted $2$ units to the right and up $3$ units.

Notice that the domain remains $(-\infty,\infty)$, but the range has been shifted. For $g(t)$ the output values never get smaller than $3$, so the range is $[3,\infty)$.

$h(x) = -3|x+1|$ is a flip over the $x$-axis, a vertical stretch by a factor of $3$ and a shift left by $1$ unit.

Once again, the domain is unchanged, all real numbers may be used for inputs. (Is there a pattern here?) The output values are all below the $x$-axis and so the range is $(-\infty,0]$.

You should be adept at working with absolute value equations by this time, but a little more practice may be helpful.

Example 3A-2: Solving absolute value equations

  1. Solve: $| 2x + 1| = 5$

    Think of this as $| \mbox{whatever} | = 5$ and the "whatever" between the absolute value bars can be either $5$ or $-5$. This gives two equations to solve. The work is shown side-by-side below (even though you probably can't work on both at the same time):

    $2x + 1 = 5$ or $2x + 1 = -5$
    $2x = 4$ or $2x = -6$
    $x = 2$ or $x = -3$

  2. Solve: $4 - | x - 3| = -10$

    We know how to deal with an absolute value expression that is equal to a number, so we will rewrite this equation in a form we know how to deal with: $$4 - |x-3|=-10$$ $$-|x-3| = -14$$ $$|x-3|=14$$ Now, solve this the "usual" way:

    $x - 3 = 14$ or $x - 3 = -14$
    $x = 17$ or $x = -11$

    3. The key idea is to isolate the absolute value on one side and then set up two equations to solve.

  3. Solve: $\dfrac{|1-x|}{5} = -2$

    Notice that when we rewrite the equation to isolate the absolute value (by multiplying both sides by $5$) we get $|1-x| = -10$. Since an absolute value does not output a negative value, there can be no solutions to the original equation.

$\S$ 3A.2 Piecewise Functions

What do you get if you put a nonnegative number into an absolute value?
Answer: you get that same number back: $| 0 | = 0$, $| 1 |= 1$, $\dots$ And so on.

We can say, as a generalization, that $| x |= x$; but only if $x$ is nonnegative.

OK, so, what happens if you put a negative number into an absolute value?
Answer: you get that same number back, but made positive.

How do you make a negative number positive? As mentioned above, you multiply it by $–1$:$| -5 |= -(-5) = 5$. We can say, as a generalization, that $| x |= -x$. but only if $x$ is negative.

So the absolute value function can be defined in two pieces:

$$| x |=\left\{ \begin{array}{ccc} x , & x \geq 0 \\ -x , & x \lt 0 \end{array} \right.$$

If you’ve never seen this before, it looks extremely odd. If you try to pin that feeling down, I think you’ll find this looks odd for some combination of these three reasons.

  1. The whole idea of a “piecewise function”—that is, a function which is defined differently on different domains—may be unfamiliar. Think about tax brackets when you file your Federal Income tax form. Incomes that fall in one bracket pay a different income rate than in another bracket. So, in fact, functions defined in this “piecewise manner” are more common than you might think.

  2. The $-x$ looks suspicious. “I thought an absolute value could never be negative!” Well, that’s right. But if $x$ is negative, then $-x$ is positive. Instead of thinking of the $-x$ as “negative $x$” it may help to think of it as “change the sign of $x$.”

  3. Even if you get past those objections, you may feel that we have taken a perfectly ordinary, easy to understand function, and redefined it in a terribly complicated way. Why bother?

    4. Surprisingly, the reason piecewise functions were "dreamed up" is because in many situations, they are the best model for the situation--like tax brackets. Sometimes when you order items from a catalog, the price of an item decreases if you buy two or more items at a time. A function which models this situation could look like: $$\mbox{Cost of a t-shirt}=\left\{ \begin{array}{ccc} 10 , & 1 \leq x \leq 2 \\ 9.50 , & 3 \leq x \leq 4 \\ 8, & x \geq 5 \end{array} \right.$$ For this catalog, you pay $\$10$ if you order one or two t-shirts, but the price per t-shirt drops to $\$9.50$ if you order three or four and to $\$8$ if you order five or more.

At this point, you should try working with piecewise functions to feel more comfortable with them.

Example 3A-3: Using a piecewise function

Find $V(0)$ and $V(5)$ where $$V(t) =\left\{ \begin{array}{ccc} t^2+1, & t \leq 2 \\ 8, & t \gt 2 \end{array} \right.$$ Since $0<2$, we use the first piece of the function to find $V(0) = 0^2+1 = 1$.
The second piece is used to find $V(5)$ since $5>2$, In this case, $V(5) = 8$.

Example 3A-4: Graphing a piecewise function

Graphing a piecewise functions is pretty much like graphing an absolute value function. You just graph each piece of the function on the interval for which it is defined. Let's graph $$f(x) =\left\{ \begin{array}{ccc} x+7 , & x \leq -2 \\ x^2 + 1 , & -2 \lt x \leq 1 \\ 3 , & x \gt 1 \end{array} \right.$$ First visualize where the three pieces will be plotted. The domains of the "pieces" start or end at $x=-2$ and $x=1.$

For values of $x$ smaller than $-2$ $f(x)$ is defined by $x +3$, which is a line with slope $1$ and $y$-intercept $3$. So, we graph this line in it's domain--to the left of $-2$. At $x = -2$, $y=-2+7=-5$: plot the point $(-2,-5)$. Now, count off the slope by moving to the left of this point: one unit down to one unit going left which is $\frac{-1}{-1} = 1$. Draw the line.
