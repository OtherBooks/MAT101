MAT-101: WeBWorK Quick Reference

Real Numbers

  • It is generally a good idea to maintain \(3\) to \(5\) decimals of accuracy when submitting answers in WeBWorK
  • Type 3/7 for \( \dfrac{3}{7} \)
  • Type .375 for \( .375 \)
  • Type 7.25% for \(7.25\% \)
  • 

Arithmetic Operations

  • Type x^2 for \( x^2 \)

Set Operations

  • Type {1,4,5,6,7,8} for \(\{1,4,5,6,7,8 \} \) in WeBWorK. Elements of sets are contained in curly braces.
  • Type capital U for \(\cup \) the union symbol
  • Type (-inf,-2]U[1,inf) for \((-\infty, -2]\cup [1,\infty)\)
  • Type None for empty sets

Intervals

  • Type [4,inf) for the interval \( [4,\infty )\)
  • Type (-inf,-4) for the interval \( (-\infty, -4)\)
  • Type (-inf,inf) for the interval \( (-\infty, \infty)\)

Inequalities

  • Type the two symbols > = together for \(\geq\)
  • Type the two symbols < = together for \(\leq\)
  • Type -inf < x < inf for the inequality \(-\infty \lt x \lt \infty\)
  • Type x < = -3 for the inequality \(x \leq -3\)

Equations

  • 2 solutions: Type x=7 or x=-7
  • 0 solutions: Type No solution

Graphs and Lines

  • Points in the plane are entered as (#,#)=(horizontal #, vertical #)=(independent, dependent)=(predictor, response)
  • For the first quadrant, type Q1, for the second quadrant, type Q2, etc.
  • Type undefined for the slope of a vertical line in WeBWorK
